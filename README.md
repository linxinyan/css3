# css3
css3 learning note

* 一、浏览器私有前缀：
* 私有前缀	对应的浏览器
* -webkit-	Chrome和Safari
* -moz-	    Firefox
* -ms-	    IE
* -o-	      Opera
* 更多CSS3属性在各大浏览器的兼容性情况，在https://caniuse.com/上查看

* 二、选择器
* 1、属性选择器
* 选择器	说明
* E[attr^=“xxx”]	选择属性是以xxx开头的元素E
* E[attr$=“xxx”]	选择属性是以xxx结尾的元素E
* E[attr*=“xxx”]	选择属性包含xxx的元素E
